Pendahuluan

Sistem Informasi Tata Naskah Dinas (SiTanda) adalah sebuah sistem informasi yang bertujuan untuk menyimpan dan mengelola semua naskah dinas yang berupa surat masuk dan surat keluar yang ada pada suatu instansi. Berikut ini adalah deskripsi arsitektur jaringan Sistem Informasi Tata Naskah Dinas .

 Gambar Arsitektur Sistem SiTanda

Secara umum sistem SiTanda terdiri atas beberapa bagian yaitu

  1. Server SiTanda adalah komputer induk yang menjalankan sistem SiTanda dan menyimpan data aplikasi ke dalam database.
  2. Aplikasi SiTanda adalah aplikasi berbasis web yang bertugas menyimpan data naskah dinas berupa surat masuk ke dalam database dan menampilkan ke pengguna melalui aplikasi mobile atau halaman web.
  3. Operator SiTanda bertugas memindai naskah dinas berupa surat masuk menjadi berbentuk file pdf dan kemudian memasukkan data naskah dinas tersebut ke dalam Sistem SiTanda melalui aplikasi web. Operator SiTanda kemudian mendisposisi naskah dinas tersebut ke pegawai yang bertanggung jawab.
  4. Pegawai yang bersangkutan bisa melihat data surat yang masuk tersebut melalui aplikasi web atau melalui aplikasi mobile. Pegawai dapat mendisposisi atau menutup naskah dinas tersebut
  5. Pegawai juga bisa membuat surat dari template yang telah dibuat oleh operator SiTanda

Petunjuk Penggunaan

Aplikasi Web Sistem Informasi Tata Naskah Dinas (SiTanda)

untuk Operator SiTanda (admin)

Operator SiTanda harus login ke dalam aplikasi web SiTanda melalui halaman admin dengan alamat http://sitanda.iain-tulungagung.ac.id/admin/login menggunakan username dan password yang telah disediakan. Berikut adalah tugas tugas yang dapat dilakukan oleh operator SiTanda.

  1. Mendaftarkan pegawai ke dalam sistem SiTanda, pegawai tersebut harus terlebih dahulu terdaftar di aplikasi SIAKAD yang telah ada.
  2. Menentukan alur disposisi setiap pegawai
  3. Memasukkan data naskah dinas ke dalam sistem SiTanda
  4. Mendisposisi naskah dinas ke pegawai
  5. Membuat template surat keluar

Berikut ini akan dijelaskan lebih detail dari langkah-langkah diatas.

Mendaftarkan pegawai ke dalam sistem SiTanda

Untuk mendaftarkan pegawai ikutilah langkah langkah berikut

  1. Login ke halaman admin
  2. Klik menu Daftar Jabatan
  3. Pada halaman Daftar Jabatan klik tombol Tambah Jabatan
  4. Pada halaman Tambah Jabatan isi semua field berikut:
  • Atasan
  • Nama Jabatan
  • Nama Staff        
  • Phone
  • Email
  • NIP

Nama pegawai diambil dari data yang ada pada siakad, dan field phone, email dan NIP akan terisi secara otomatis tetapi masih dapat diganti

  1. Klik tombol Save untuk menyimpan data

Gambar Form Tambah Daftar Jabatan

Setelah data berhasil disimpan akan tampil di tabel Daftar Jabatan. Anda dapat mengubah data yang sudah ada dengan menekan tombol Edit atau menghapus jabatan yang sudah ada dengan menekan tombol Delete pada kolom Action.

Gambar Tabel Daftar Jabatan

Setelah anda berhasil mendaftarkan pegawai ke sistem SiTanda maka anda bisa mengatur alur disposisi pegawai tersebut.

Untuk melihat gambar struktur organisasi dari jabatan yang telah dimasukkan silahkan klik tab Chart sedangkan untuk melihat alur disposisi klik tab Alur Disposisi

Menentukan alur disposisi setiap pegawai

Untuk menentukan alur disposisi pada setiap pegawai ikuti langkah berikut:

  1. Login ke halaman admin
  2. Klik menu Daftar Jabatan
  3. Pada halaman Daftar Jabatan klik tombol Edit pada pegawai yang ingin diatur alur disposisinya
  4. Kemudian anda akan masuk pada halaman Update Jabatan, pada halaman ini tekan tab Alur Surat
  5. Pada halaman Alur Surat, pilih tujuan pengiriman pada dropdown Kirim Ke kemudian tekan tombol Save
  6. Anda dapat melihat tujuan disposisi pegawai tersebut pada tabel Daftar Tujuan

Template Surat Dinas

Operator SiTanda dapat membuat template surat dinas yang nanti dapat dipakai oleh pegawai pada halaman aktivitas disposisi. Untuk membuat template surat diperlukan file Microsoft Word dalam format docx. Sehingga nanti pada halaman aktivitas disposisi ketika user membuat surat dinas baru,  aplikasi SiTanda akan menggunakan file template ini dan mengganti setiap text yang diawali dan diakhiri dengan kurung kurawal dengan input yang diterima dari user dan kemudian menyimpan file tersebut ke dalam database. Berikut adalah contoh file template dan form yang secara otomatis dibuat pada halaman aktivitas disposisi.

Gambar contoh file docx untuk template surat dinas

Gambar form yang otomatis dibuat dari file docx

Membuat template surat dinas

Berikut adalah langkah untuk membuat template surat dinas

  1. Login ke halaman admin
  2. Klik menu Template > Surat Dinas
  3. Pada halaman Daftar Jenis Surat Dinas klik tombol Tambah Data kemudian akan muncul halaman Tambah Jenis Surat Dinas
  4. Masukkan Nama jenis surat dan pada field Template File pilih file yang akan dijadikan template surat keluar, pada field Default Tanda Tangan pilih tanda tangan yang akan digunakan secara default pada surat keluar ini, pada field Petunjuk Pengisian masukkan cara pengisian form yang benar
  5. Tekan tombol Simpan untuk menyimpan template surat dinas

Gambar Form Tambah Jenis Surat Dinas

Membuat template tanda tangan

Berikut adalah langkah untuk membuat template tanda tangan

  1. Login ke halaman admin
  2. Klik menu Template > Daftar Tanda Tangan
  3. Pada halaman Daftar Tanda Tangan klik tombol Tambah Data kemudian akan muncul halaman Form Tanda Tangan
  4. Masukkan Nama pemilik tanda tangan dan Isi Text dengan tanda tangan

Pesan Masuk Mahasiswa

Halaman ini berisi surat yang telah dibuat oleh mahasiswa. Untuk mendownload surat tersebut tekan link pada kolom Ref No atau Nama File. Surat tersebut dapat juga diubah oleh admin atau didisposisikan kepada pegawai lain.

Untuk mengubah atau mendisposisi surat tersebut tekan icon edit pada kolom Action. Setelah itu akan muncul halaman Isian form. Untuk mengubah isi dari surat tersebut silahkan klik tab Detail dan ubah field yang diperlukan, kemudian tekan tombol Simpan.

Untuk mendisposisi surat ke pegawai lain silahkan klik tab Disposisi kemudian pada bagian Disposisi Selanjutnya, pada field Disposisi Ke pilih staf yang dituju dan pada field Petunjuk pilih satu atau lebih petunjuk. Pada field Catatan isi dengan catatan untuk pegawai yang akan menerima disposisi, kemudian tekan tombol Simpan. Surat tersebut kemudian akan diterima oleh staf yang dituju pada menu Formulir Mhs

Aplikasi Web Sistem Informasi Tata Naskah Dinas (SiTanda)

untuk Pegawai

Pegawai harus login ke dalam aplikasi web SiTanda melalui halaman user dengan alamat http://sitanda.iain-tulungagung.ac.id/user/login menggunakan username yaitu NIP dan password yang telah disediakan. Setelah berhasil login sebagai pegawai anda akan masuk ke halaman dashboard berisi daftar pesan yang ditujukan kepada anda. Untuk melihat detail dari pesan yang masuk klik link di kolom No Surat.

Gambar Daftar Surat Masuk

Kemudian akan muncul halaman detail pesan yang berisi panel Detail Surat Masuk dan tab Disposisi dan tab Surat Keluar.

Gambar Detail Surat Masuk

Pada tab Disposisi, pegawai dapat mendisposisi surat ke pegawai lain, pada tab Surat Keluar pegawai dapat membuat surat keluar.

Mendisposisi surat ke pegawai lain

Untuk mendisposisi surat ke pegawai lain ikutilah langkah-langkah berikut :

  1. Login ke halaman pegawai
  2. Pada halaman dashboard atau daftar pesan klik link pada kolom No Surat untuk masuk ke halaman detail surat
  3. Pada halaman detail surat, pada tab Disposisi > Disposisi Selanjutnya pilih jabatan dari pegawai yang akan dikirimi disposisi pada field Disposisi Ke dan pada field Petunjuk pilih petunjuk yang sesuai. Masukkan catatan pada field Catatan
  4. Klik tombol Simpan untuk mengirim disposisi
  5. Apabila terjadi kesalahan dan ingin menghapus disposisi anda bisa menekan tombol Hapus pada Riwayat Disposisi

Gambar Form Disposisi

Membuat surat keluar

Untuk membuat surat keluar ikutilah langkah-langkah berikut :

  1. Login ke halaman pegawai
  2. Pada halaman dashboard atau daftar pesan klik link pada kolom No Surat untuk masuk ke halaman detail surat
  3. Pada halaman detail surat, klik tab Surat Keluar
  4. Pada form Buat Surat Baru pilih Jenis Surat dan tekan tombol Buat

Gambar Form Buat Surat Baru

  1. Kemudian akan muncul dialog yang diperlukan untuk membuat jenis surat tersebut. Isi semua field yang diperlukan, dan field Nama Dokumen
  2. Tekan tombol Proses untuk membuat surat keluar berdasarkan data yang dimasukkan sebelumnya.

Menyelesaikan Surat

Surat yang telah dianggap selesai dapat ditutup sehingga tidak bisa didisposisikan lagi. Berikut langkah untuk menutup atau menyelesaikan surat :

  1. Login ke halaman pegawai
  2. Pada halaman dashboard atau daftar pesan klik link pada kolom No Surat untuk masuk ke halaman detail surat
  3. Pada halaman detail surat, klik tombol Selesai yang ada pada sebelah kanan atas halaman
  4. Kemudian akan muncul dialog Kesimpulan, isi field kesimpulan dari surat
  5. Tekan tombol Save untuk menyimpan kesimpulan.  

Gambar Form Kesimpulan

Formulir Mahasiswa

Surat yang dibuat oleh mahasiswa dan didisposisikan ke pegawai oleh admin dapat dilihat oleh pegawai pada menu Pesan Masuk Mahasiswa. Untuk mengubah atau mendisposisi surat tersebut tekan icon edit pada kolom Action. Setelah itu akan muncul halaman Isian form. Untuk mengubah isi dari surat tersebut silahkan klik tab Detail dan ubah field yang diperlukan, kemudian tekan tombol Simpan.

Untuk mendisposisi surat ke pegawai lain silahkan klik tab Disposisi kemudian pada bagian Disposisi Selanjutnya, pada field Disposisi Ke pilih staf yang dituju dan pada field Petunjuk pilih satu atau lebih petunjuk. Pada field Catatan isi dengan catatan untuk pegawai yang akan menerima disposisi, kemudian tekan tombol Simpan. Surat tersebut kemudian akan diterima oleh staf yang dituju pada menu Formulir Mhs

Aplikasi Mobile Sistem Informasi Tata Naskah Dinas (SiTanda)

untuk Pegawai

Aplikasi mobile ini hanya ditujukan kepada user dengan role pegawai saja. Langkah pertama yang perlu dilakukan adalah mendownload aplikasi SiTanda dari https://play.google.com/store/apps/details?id=com.aldoapp.iainta dan menginstallnya pada smart phone anda. Pada saat pertama kali menjalankan aplikasi akan muncul dialog yang meminta persetujuan anda untuk mengakses contact pada smart phone anda. Tekan tombol Allow untuk melanjutkan.

Gambar android permission dialog

Setelah itu akan tampil halaman login.

Halaman Login

Masukkan username dan password anda dan tekan tombol Login

Gambar halaman login

Setelah berhasil login anda akan melihat tampilan halaman Daftar Pesan yang berisi sejumlah pesan masuk diurutkan berdasarkan tanggal pembuatan pesan.

Halaman Daftar Pesan

Gambar halaman daftar pesan masuk

Halaman pesan berisi pesan yang ditujukan kepada pengguna, data yang ditampilkan antara lain:

  • Judul Surat
  • Perihal surat
  • Tanggal acara
  • Sifat surat
  • Tanggal pesan dibuat
  • Pegawai yang mendisposisi

Pada halaman daftar pesan anda juga dapat mencari pesan dan menyaring pesan berdasarkan pesan yang telah dibaca atau pesan yang belum dibaca. Untuk melihat detail pesan anda bisa mengklik pesan tersebut sehingga akan muncul halaman detail pesan.

Halaman Detail Pesan

Halaman Detail Pesan terdiri atas beberapa tab yaitu: Detail Surat, Aktivitas dan Docs. Tab Detail Surat berisi data mengenai surat masuk, tab Aktivitas berisi aktivitas disposis yang terjadi pada surat masuk tersebut, dan tab Docs berisi naskah dinas yang telah dibuat berkaitan dengan dengan surat masuk tersebut.

Gambar halaman detail pesan

Pada tab Detail Surat anda dapat melihat data tentang surat masuk tersebut dan dapat mendownload hasil scan dari surat masuk tersebut melalui link Download PDF. Anda dapat juga menyelesaikan surat dengan menekan tombol Selesai.

Anda dapat juga melihat aktivitas disposisi surat masuk tersebut pada tab Aktivitas.

Gambar tab aktivitas

Untuk mendisposisi surat masuk ke pegawai yang lain anda bisa menekan tombol bergambar amplop yang berada pada kanan bawah halaman. Kemudian akan muncul dialog yang berisi form tujuan disposisi. Isi semua field: Disposisi Ke, Petunjuk, Catatan kemudian tekan OK untuk mengirim disposisi tersebut.

Gambar dialog form disposisi

Disposisi yang telah berhasil disimpan akan muncul pada tab Aktivitas

Gambar aktivitas disposisi

Pegawai yang menerima disposisi tersebut akan mendapat notifikasi pada layar smart phonenya.

Gambar notifikasi pada layar smart phone

Anda dapat juga melihat surat keluar yang telah dibuat untuk surat masuk tersebut pada tab Docs. Anda dapat mendownload file tersebut ke smart phone dengan mengklik tombol … pada item tersebut kemudian akan muncul pop up menu, kemudian klik Download.

Gambar tab Docs

Untuk membuat naskah dinas baru anda dapat memilih jenis surat pada bagian bawah layar dan tekan tombol Buat Surat. Kemudian akan muncul dialog parameter dokumen seperti gambar berikut.

Gambar dialog parameter dokumen

Silahkan isi semua field yang diperlukan dan field Nama Dokumen adalah field yang wajib diisi. Kemudian tekan tombol OK. Beberapa saat kemudian naskah dinas tersebut akan muncul pada halaman Docs.

Anda dapat mengubah naskah dinas yang telah ada dengan menekan tombol ... pada item dan setelah muncul pop up menu kemudian klik Edit. Setelah muncul dialog parameter dokumen seperti langkah sebelumnya, anda bisa mengubah isi field yang diperlukan dan menekan tombol OK untuk menyimpan perubahan.

 Petunjuk Penggunaan

Aplikasi Web Sistem Informasi Tata Naskah Dinas (SiTanda)

untuk Mahasiswa

Mahasiswa dapat membuat membuat surat yang ditujukan kepada staf bagian Akademik dengan cara login pada http://sitanda.iain-tulungagung.ac.id/student/login terlebih dahulu. Setelah login kemudian klik tombol Daftar Formulir untuk membuat formulir. Pada halaman Daftar Formulir klik Isi Form pada formulir yang diinginkan.

Gambar Daftar Formulir

Setelah mahasiswa mengklik salah satu formulir kemudian akan muncul halaman Isian form yang berisi field-field sesuai dengan formulir yang dipilih. Isi semua field yang diperlukan dan field yang wajib diisi adalah Catatan dan Nama Dokumen (*.docx) kemudian tekan tombol Simpan

Setelah data berhasil disimpan kemudian akan muncul halaman My Documents yang berisi daftar surat-surat yang telah dibuat sebelumnya. Untuk mendownload file surat dalam bentuk docx silahkan tekan link pada kolom Nama File atau Ref No. Untuk mengubah isi dari parameter dari surat yang telah dibuat, tekan link Edit pada kolom Action
