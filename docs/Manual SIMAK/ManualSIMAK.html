SEKILAS PANDANG

Aplikasi SIMAK adalah Sistem Informasi yang mengelola data kepegawaian mulai dari data jenis pegawai, bio data pegawai, dan lain-lain. Setiap pegawai yang terdaftar dapat masuk ke dalam aplikasi SIMAK dan membuat proposal untuk perubahan data kepegawaian. Proposal tersebut kemudian akan diperiksa oleh admin dan admin dapat menyetujui proposal tersebut apabila data yang diajukan valid. Data kepegawaian akan diubah berdasarkan proposal yang disetujui tersebut.

Dalam aplikasi SIMAK terdapat dua modul:

  1. Modul Admin

Modul ini dibuat untuk membuat pegawai baru, melihat semua pegawai yang terdaftar dalam sistem, serta mengubah semua data pegawai yang ada. Alamat login untuk admin adalah http://simak.uinsatu.ac.id/admin . login yang dipakai adalah alamat email admin

  1. Modul Pegawai

Module ini dibuat untuk pegawai yang terdaftar dalam sistem ini agar bisa mengubah data pribadinya sendiri. Alamat login untuk module ini adalah http://simak.uinsatu.ac.id/staff/login . login yang dipakai adalah alamat email pegawai.

Alur Aplikasi

  1. Admin aplikasi SIMAK mendaftarkan pegawai ke dalam sistem
  2. Pegawai yang terdaftar kemudian membuat proposal data kepegawaian
  3. Pegawai mengajukan proposal data kepegawaian
  4. Admin mengecek proposal data kepegawaian dan menyetujui proposal tersebut apabila data proposal tersebut valid atau menolak proposal tersebut apabila data tidak valid dan memberikan alasan penolakan
  5. Pegawai dapat mengubah proposal yang telah ditolak dan mengajukan lagi proposal tersebut ke admin
  6. Data kepegawaian akan berubah sesuai dengan data dari proposal yang disetujui

MODUL ADMIN

Modul admin terdiri atas beberapa menu diantaranya:

Task

  • Daftar Staff

Berisi daftar pegawai yang ada pada sistem informasi SIMAK

  • Tambah Staff

Digunakan untuk menambahkan pegawai baru ke dalam sistem informasi SIMAK

Untuk mengubah data seorang pegawai admin harus masuk ke halaman Daftar Staff (Gambar 2.1) kemudian menekan tombol Edit pada pegawai yang diinginkan. Setelah menekan tombol Edit, Admin akan berpindah ke halaman Profile dari pegawai yang terpilih.

Gambar 2.1 Halaman Daftar Staff

Setelah itu akan muncul menu-menu baru dibawah menu Task dan admin bisa mengubah tiap data dari pegawai yang terpilih tersebut melalui menu tersebut. Menu-menu baru tersebut pada dasarnya sama dengan menu yang ada pada modul Staff.

Merubah Jenis Pegawai

Pada aplikasi SIMAK terdapat 2 jenis pegawai yaitu:

  1. Tenaga Pendidik
  2. Tenaga Kependidikan

Tenaga Pendidik dibagi lagi menjadi beberapa kategori yaitu:

  1. PNS
  2. CPNS
  3. DOSTAP BUKAN PNS
  4. DLB
  5. PPPK

Tenaga Kependidikan dibagi lagi menjadi beberapa kategori yaitu:

  1. PNS
  2. CPNS
  3. KONTRAK
  4. PPPK

Untuk merubah jenis pegawai tekan tombol edit pada panel Bio Data (Gambar 2.2) pada halaman Profile pegawai

Gambar 2.2 Panel Bio Data

Kemudian akan muncul Panel Perubahan Jenis Pegawai (Gambar 2.3). Silahkan pilih Jenis Pegawai dan Kategori yang sesuai kemudian tekan tombol Save.

Gambar 2.3 Panel Perubahan Jenis Pegawai

Merubah Data Kepegawaian

Admin bisa secara langsung mengubah data pegawai yang terpilih lewat tab Form atau dengan cara menyetujui proposal pengajuan yang telah diajukan dari pegawai yang bersangkutan lewat tab Pengajuan (Gambar 2.4) kemudian menekan tombol Edit/Review pada kolom Action

Gambar 2.4 Tab Pengajuan

Setelah menekan tombol Edit/Review, Admin akan diarahkan ke halaman form pengajuan perubahan data kepegawaian seperti pada Gambar 2.5. Pada halaman ini Admin harus memeriksa data yang diajukan tersebut apakah sudah benar atau belum.

Gambar 2.5 Form Pengajuan

Apabila terdapat kesalahan atau data yang diisi kurang maka Admin dapat menolak pengajuan tersebut dengan menekan tombol Reject. Kemudian akan muncul dialog Tolak Pengajuan (Gambar 2.6).  Silahkan masukkan alasan mengapa pengajuan tersebut ditolak kemudian tekan tombol Save.

Gambar 2.6 Dialog Tolak Pengajuan

Pengajuan yang telah ditolak oleh Admin Kepegawaian dapat diperbaiki lagi oleh pegawai yang bersangkutan dan diajukan lagi apabila telah siap.

MODUL PEGAWAI

Modul pegawai terdiri atas beberapa menu diantaranya:

Profile

Halaman Profile (Gambar 3.1) berisi ringkasan data pegawai yang login pada aplikasi SIMAK yaitu data Bio Data, Alamat, Kepegawaian, Inpassing, Jabatan Fungsional Kepangkatan.

Gambar 3.1 Halaman Profile

1.1 Biodata

Pada tab Bio Data, seorang pegawai dapat merubah photo pegawai, merubah password untuk login dan juga merubah data diri lainnya.

Untuk mengubah password pegawai, tekan tombol Ganti Password dan untuk mengubah gambar avatar bisa dilakukan dengan menekan gambar avatar yang ada pada panel Bio Data.

Pegawai bisa mengajukan perubahan data Bio Data lewat form Perubahan Bio Data (Gambar 3.2)

Gambar 3.2 Tab Form Perubahan Bio Data

Pada halaman Perubahan Bio Data terdapat kolom Data Lama Disetujui dan Data Baru. Kolom Data Lama Disetujui akan kosong apabila belum ada data yang disetujui.

Untuk mengubah bio data pegawai silahkan masukkan data yang baru untuk : Nama, NIK, Jenis Kelamin, Tempat Lahir, Tanggal Lahir, Nama Ibu, Agama, Status Perkawinan, Email, Username kemudian tekan tombol Save. Pengajuan yang telah dibuat sebelumnya akan disimpan di tab History (Gambar 3.3).

Gambar 3.3 Tab History Perubahan Bio Data

Untuk mengubah data pengajuan yang ada tekan tombol Edit pada kolom Action. Data perubahan bio data yang dianggap telah siap dapat diajukan kepada Admin Kepegawaian dengan menekan tombol Ajukan pada kolom Action. Apabila Admin Kepegawaian menyetujui perubahan bio data tersebut maka data tersebut akan muncul di bagian Data Lama Disetujui.

1.2 Alamat

Pada Alamat, seorang pegawai dapat merubah detail alamat tempat tinggal pegawai. Pegawai bisa mengajukan perubahan data Alamat lewat form Perubahan Alamat (Gambar 3.4)

Gambar 3.4 Tab Form Perubahan Alamat

Pada halaman Perubahan Alamat terdapat kolom Data Lama Disetujui dan Data Baru. Kolom Data Lama Disetujui akan kosong apabila belum ada data yang disetujui.

Untuk mengubah alamat pegawai silahkan masukkan data yang baru untuk : Alamat, RT, RW, Dusun, Desa/Kelurahan, Kecamatan dan Kode Pos pada halaman Perubahan Alamat, kemudian tekan tombol Save. Pengajuan yang telah dibuat sebelumnya akan disimpan di tab Pengajuan (Gambar 3.5).

Gambar 3.5 Tab Pengajuan Alamat

Untuk mengubah data pengajuan yang ada tekan tombol Edit pada kolom Action. Data perubahan alamat pegawai yang dianggap telah siap dapat diajukan kepada Admin Kepegawaian dengan menekan tombol Ajukan pada kolom Action kemudian status pengajuan akan berubah dari Draft menjadi Diajukan. Apabila Admin Kepegawaian menyetujui perubahan alamat tersebut maka data tersebut akan muncul di bagian Data Lama Disetujui.

1.3 Kepegawaian

Pada form Perubahan Kepegawaian terdapat kolom Data Lama Disetujui dan Data Baru. Kolom Data Lama Disetujui akan kosong apabila belum ada data yang disetujui.

Untuk mengubah data kepegawaian pegawai silahkan masukkan data yang baru untuk : Pangkat, Jabatan, NIP, Nomor SK CPNS, SK CPNS Mulai Tanggal, Nomor SK TMMD, Tanggal Mulai Menjadi Dosen, Sumber Gaji, NIDN, Gelar Depan, Gelar Belakang, KGB Selanjutnya, KP Selanjutnya, Tanggal TMT CPNS, Tanggal TMT PNS, Status, Masa Kerja, Tanggal Pensiun, Pendidikan Terakhir kemudian tekan tombol Save. Pengajuan yang telah dibuat sebelumnya akan disimpan di tab Pengajuan (Gambar 3.6).

Gambar 3.5 Tab Form Perubahan Kepegawaian

Gambar 3.6 Tab Pengajuan Kepegawaian

Apabila Admin Kepegawaian menyetujui perubahan kepegawaian tersebut maka data tersebut akan muncul di bagian Data Lama Disetujui.
